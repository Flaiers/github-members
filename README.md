How to start my project
-----------------------
#### First, run the script install.sh

> Для начала запустите скрипт install.sh

```bash
./install.sh
```

If you get an error, run:

> Если у вас возникла ошибка, запустите:

```bash
sudo chmod a+x install.sh
```
